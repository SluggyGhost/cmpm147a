Experiment 1 - Hello Glitch/Generator

The next day at the tavern

Description

Create a unique generator on glitch.com and move the working generator to my portfolio page.

Technical

A grammar is defined with simple replacement tokens. A replacer() function picks a random replacement for any token that appears in the text.

Reflection

Translating the code from glitch.com to a static webpage was difficult to figure out, even with the professor's example. I feel like I should have started working on this from the second it was available, especially given the unexpected computer troube.
